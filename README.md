## You might be wondering, how do you play this game?

Below is a detailed step-by-step walkthrough of how to play the game + run the code.
<br/> 

### First, how to run the program:

1. Save/copy the code. 
2. Create a new file (name it something simple that represents the code like simpsons-game.py).
3. Open your program/terminal and open the file you created.
4. Make sure pygame is intstalled. 
4. Press the play icon (▶️) to run the game in the terminal!
<br/>

##  How to run/play the game after opening the program:

1. 👋 | Move Homer Simpson by pressing your left, and right arrow keys to move left and right, and your spacebar to jump. 

2. Navigate to the donuts be moving along the ground, and jumping on the bushes!

3. 🤔 | The goal of the game is to collect all 5 donuts! You can see your donut colllection progress in the top left corner of your game. 

4. ‼️ | If you succesfully collected all five donuts, than congrats, you won the game and will be presented with a "You Won!" message!
