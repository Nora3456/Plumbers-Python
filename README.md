## You might be wondering, how do you play this life decision simulator?

Below is a detailed step-by-step walkthrough of how to play the life decision simulator + run the code.
<br/> 

### First, how to run the program:

1. Save/copy the code. 
2. Create a new file (name it something simple that represents the code like life_decision_simulator.py).
3. Open your program/terminal and open the file you created.
4. Press the play icon (▶️) to run the script in the terminal!
<br/>

##  How to run/play the simulator after opening the program:

1. 👋 | The code will present you with a welcome message, and give you a quick note about what to press for each decision, for instance if you have 3 decisions, you should input either "1", "2", or "3". 

2. 🤔 | The code asks you to pick your first adventure, you have 2 options. Could this adventure lead to your destiny... let's find out! 

-‼️ **REMEMBER** ‼️ | The adventure choices will start with "1;", "2:", (or sometimes) "3:". Depending on which adventure you would like to pick, type in the coresponding number to the adventure in your terminal, and then press enter.

3. 🧐 | You will than go through a sequence of adventures to choose from. Choose every adventure wisely, you never know where you might end up! If you are confused on how to input your decision, look back at the *"REMEMBER"* reminder above!

**🧑‍🏫 |  EXAMPLE** (From the code!):
- Here's the code example: *"You now have the permanent ability to 1; time travel or 2: shapeshift:"*. If you wish to have the permanent ability to time travel, type "1" in your terminal, and then press enter.If you wish to have the permanent ability to shapeshift, type "2" in your terminal, then press enter. After pressing a new message will be displayed with more adventures + choices!

3. 🎉 | You made it to the end..? If you have gone through all your choices, and there is no option to input another answer (or no more questions appear)... **Congrats**, your destiny will be read to you!

### 😬 Didn't like your destiny? 😥
**So you didn't like your destiny... well this is awkward.** 
<br>
Anyways... I get it, you were hoping for a different outcome, well don't worry! You can run the code again, by changing **just *1*** adventure choice differently from your choices on your previous simulator, your destiny can be changed! 
...Sure some may say "it's fate" and "you can't change your destiny", well I say you can!
